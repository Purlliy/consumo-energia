## Projeto Consumo de Energia

---

## Sobre o projeto:
O projeto se chama "Consumo-energia" e o objetivo dele é ajudar usuários estimar quanto um aparelho gasta de energia elétrica por mês. Utilizando de uma fórmula que requer potência e tempo de uso diário para calcular a estimativa.

---

## Sobre o programa:
O programa utilizada da fórmula "consumoMensal = (potencia * horasDia * 30) / 1000", para calcular o  consumo mensal e "custoEstimado = consumoMensal * 0.75" para calcular o custo estimado do aparelho. 

Para utilizar o programa, primeiro deve ser digitado o nome do aparelho que quer calcular o custo e consumo (ex.: Geladeira, ventilador, computador), após ser digitado o nome do aparelho, deve ser inserido a potência do aparelho em watts (W). E por ultimo se pede para colocar o tempo médio diário de uso do aparelho em horas. Após ter digitado todas essas informações, o programa irá entregar o nome do aparelho, o consumo mensal do aparelho e o custo mensal dele.
